# spotify-projects
Différents projets avec l'API Spotify.

MAJ 2025: Malheureusement, Spotify a mis à jour sa politique d'API et de nombreuses features sont devenues dépréciées, ce qui entraîne le mauvais fonctionnement voir l'incapacité de ces projets, en attendant un hypothétique changement.

- spotifyWeekly.py : Création d'une application qui va sauvegarder la playlist "Discover Weekly" dans une autre playlist appelée "Saved Weekly". 

- /mood-analyzer : Application web qui récupère les tops artistes et chansons d'un utilisateur, et permet d'analyser ces écoutes.
